Iframe compatibility

Off by default. Enable only if a built-in AI service refuses to load inside the side panel.

Chrome applies these rules browser-wide while enabled, not only inside anyside. Custom URLs and main-frame browsing are never modified. Fallback window mode is the safer alternative when iframe loading is unreliable.
