Iframe compatibility

Off by default. Enabling this removes selected frame-blocking headers from subframe requests to chatgpt.com, claude.ai, gemini.google.com, www.perplexity.ai, and notebooklm.google.com while the mode is on. Chrome applies the rules browser-wide, not only inside anyside's Side Panel. Custom URLs and main-frame browsing are not covered.
